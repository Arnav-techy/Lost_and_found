# 🎵 Lost & Sound - Lost & Found Platform

A comprehensive full-stack web application that helps people report lost items and discover found items in their community. Built with the MERN stack (MongoDB, Express.js, React, Node.js).

## ✨ Features

### For Users
- **User Authentication** - Secure registration and login with JWT
- **Report Lost Items** - Post detailed descriptions of lost belongings
- **Report Found Items** - List items you've found to help return them
- **Search & Filter** - Find items by category, location, date, or keywords
- **Claim Items** - Request to claim items you believe are yours
- **User Dashboard** - Manage your reported and claimed items
- **Real-time Status Updates** - Track the status of your reports and claims

### For Admins
- **Moderate Listings** - Review and verify reported items
- **User Management** - Manage user accounts and permissions
- **Analytics Dashboard** - Track platform usage and success rates
